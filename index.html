Descrição do projeto

>Este projeto tem como objetivo abordar alguns conceitos vistos durante o módulo. Para isso, vamos criar uma Wiki e hospedá-la no GitHub Pages, utilizando a linguagem de programação Python e a manipulação de dados em formato JSON. O GitHub Pages é uma plataforma que permite hospedar sites diretamente de repositórios do GitHub, tornando-o uma escolha popular para compartilhar conteúdo, como blogs pessoais, portfólios e documentações..

Passo a passo

Primeiro, dei um fork no repositório online da turma fiz as alterações nos documentos com de markdown no VSCode abri o GitBash, dei um gitstatus para verificar se minhas alterações no VSCode foram salvas e depois de um Git Commit no projeto pelo terminal do GitBash. Depois disso, voltei ao meu repositório online do GitHub, fui em configurações e habilitei as GitHub Pages Após solicitar a criação de um token, o fiz e dei a permissão para a criação das pages pelo terminal do GitBash
